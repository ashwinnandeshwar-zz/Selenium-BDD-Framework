
Instructions:

1. Problem statement: Acceptance criteria is to test multiple conversion rate.
2. Solution is implemented using java 8,cucumber,junit,maven using Intellj IDE on window System.

3. By default scripts will run on Chrome browser, can be changed. Refer configuration.propeties file.

4. Report is available on path ".\target\cucumber-report-html\cucumber-html-reports\feature-overview.html"

Steps to execute:
a. Open command prompt
b. navigate to project path
c. run "mvn clean verify"

Above maven command will trigger the automation script.
Once the execution finish, report will generate on path mentioned in point 4.
Note: Old Report will be replaced by new.
